API Reference

Your view runs in a sandboxed iframe with access to Coral's API via fetch(). Key endpoints:

  • GET /api/sessions/live — list live sessions
  • GET /api/sessions/live/{name}/events — agent events
  • GET /api/sessions/live/{name}/tasks — agent tasks
  • GET /api/sessions/live/{name}/files — changed files
  • GET /api/board/{project}/messages/all — board messages

Context is available via window.CORAL_CONTEXT (agent name, session ID).
