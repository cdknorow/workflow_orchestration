# Claude Agent Worktree Orchestration

This is a workflow designed for people using worktrees and multiple agents. 

run the ./launch_agents <path-to-root> and it will create an agent for each worktree inside a tmux session. The status of each of your models will be displayed inside of the Dashboard.
