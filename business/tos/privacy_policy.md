## 1. What We Collect

### 1.1 Data We Collect Automatically

**License Key Validation**
When the Coral Desktop Application starts, it contacts our server to validate your license key. This request includes:
- Your license key
- A machine identifier (a hash derived from your hardware — not your MAC address or serial number)
- The software version number
- Your operating system and architecture (e.g., macOS arm64, Windows amd64)

This data is used solely to verify your license is valid and has not exceeded its activation limit. We retain validation logs for up to 90 days for abuse detection, then delete them.

**Website Analytics**
We use privacy-respecting analytics on coralai.ai to understand traffic patterns. We collect:
- Page views and referral sources
- Country-level location (no city or IP address)
- Browser type and screen size
- No cookies, no fingerprinting, no cross-site tracking

### 1.2 Data You Choose to Share (Opt-In Only)

**Crash Reports (Opt-In)**
If you enable crash reporting in the desktop application, we collect:
- Stack traces and error messages
- Software version, operating system, and architecture
- A timestamp

Crash reports do NOT include: your source code, agent output, prompts, file contents, workspace paths, or any data from your AI agent sessions.

**Usage Analytics (Opt-In)**
If you enable usage analytics in the desktop application, we collect:
- Which features you use (e.g., "launched agent team," "used sleep/wake")
- Session duration
- Number of agents run

Usage analytics do NOT include: your source code, agent output, prompts, file contents, message board content, or any data from your AI agent sessions.

### 1.3 Data We Never Collect

We do not collect, transmit, or have access to:
- Your source code or files
- AI agent prompts, instructions, or output
- Agent session content or terminal output
- Message board messages between your agents
- Git repository content, commit history, or branch names
- File paths or directory structures on your machine
- API keys or credentials for AI services (Claude, Gemini, etc.)
- Screen recordings, screenshots, or keystrokes

**Coral is a local-first application.** All agent orchestration, communication, and monitoring happens entirely on your machine.
