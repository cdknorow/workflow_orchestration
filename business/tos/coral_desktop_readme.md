## What is Coral?

Coral is your workspace for AI agents. Run Claude, Gemini, and more — with a real-time dashboard, agent-to-agent communication, and full team orchestration.

- Launch teams of AI agents with roles and behavior prompts
- Monitor every agent's status from a single dashboard
- Agents communicate through a shared message board
- Sleep and wake agents with full context preservation

## Open Source

The core Coral engine is open-source and available at [github.com/cdknorow/coral](https://github.com/cdknorow/coral). Install with:

```bash
pip install agent-coral
```

Coral Desktop builds on the open-source core with a native app experience, visual team builder, and additional Pro features.
